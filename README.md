## About The Project

Copionet Java Edition is a free plagiarism detector tool initially created to help teachers to review the student's work. It supports the main doc file types: DOCX, ODT, TXT, PDF and much more!

<br />

* [Processing]
* [Apache Tika 1.9]
* [N-Grams Library]
* [File Drop Library]
